## Customer-Facing Tone

The updated copy keeps the catalogue/enquiry model but avoids talking about internal platform state, redirects, generators, imports or database mechanics. The public framing now emphasises:

- supply, installation and floor preparation
- range and colour comparison
- stock and product enquiries
- builder/commercial project support
- practical site preparation before order or install

## Notes

HTML attributes such as `placeholder` remain because they are form input hints, not visible backend/system copy.
