# Range Page Enrichment Plan

Date: 2026-05-26

Do not apply this plan automatically to all ranges. Use Artisan Tile as the pilot model, then enrich priority ranges one supplier/category family at a time.

## Prioritisation Criteria

- Old WordPress product-category value.
- Search Console internal link count.
- Current/new product availability.
- Supplier data availability.
- Presence of real thumbnails and colour images.
- Missing specs, docs, certifications or installation notes.
- Duplicate/canonical risk.

## Artisan Tile Pilot Elements To Reuse

- Clear range description and category.
- Colour cards near the top.
- Verified colour codes where available.
- Technical specifications.
- Ratings/certifications only where verified.
- Supplier documents presented without sending customers directly to supplier sites unless approved.
- Installation/preparation notes.
- Best suited for / confirm before ordering.
- FAQ and enquiry CTAs.

## Important Policy

For Eco, HRT, Bass Timber and Preference ranges, do not link customers out to supplier websites for documents. If documents/specs cannot be hosted or converted safely, present verified information on-site and ask customers to contact Oz Timber Floor for the current supplier document/sample.
