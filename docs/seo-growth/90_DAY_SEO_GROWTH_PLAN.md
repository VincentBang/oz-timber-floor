# 90 Day SEO Growth Plan

Date: 2026-05-26

## Days 0-14: Migration Stability

- Submit final sitemap in Search Console after production cutover.
- Inspect homepage, service pages, category pages and top redirected legacy URLs.
- Monitor Page Indexing, Not Found, Page with Redirect and Duplicate canonical reports.
- Fix redirect/canonical/404 blockers only.
- Avoid broad content rewrites during volatility.
- Re-test old product aliases and old product-category URLs.
- Confirm old blog URLs resolve to matching guide pages.

## Days 15-30: Enrich Highest-Value Existing Pages

- Enrich top range pages:
  - Artisan Tile stays benchmark.
  - Villeroy & Boch Aquastop 10mm.
  - Infinite.
  - Kronoswiss Aquastop 12mm/14mm.
  - Reflections.
  - Swish Laminate.
  - Ornato Vinyl.
  - Prestige Oak.
- Restore/improve internal links in existing guide content.
- Draft first builder contractor guide.
- Improve internal links using Search Console queries.
- Add project proof where real.

## Days 31-60: Build Topic Clusters

- Publish builder contractor guide cluster.
- Publish floor preparation guide cluster.
- Add real project proof to service/category pages.
- Enrich supplier pages with stock/supply process and relevant internal links.
- Improve Google Business Profile with services, products and project photos.
- Start legitimate supplier/local partner outreach.

## Days 61-90: Expand Authority And Long-Tail Content

- Expand range/spec content for priority ranges.
- Add comparison content:
  - hybrid vs laminate
  - solid vs engineered
  - vinyl vs hybrid
  - apartment flooring checks
- Build legitimate backlinks through supplier/dealer listings, builders, fit-out partners and local platforms.
- Review Search Console keywords in positions 4-20.
- Update high-impression/low-CTR pages with better titles/meta and clearer CTAs.

## Monthly Operating Rhythm

- Search Console review.
- Improve 3 existing pages.
- Publish 1 strong guide or project asset.
- Build 1-3 legitimate local/supplier links.
- Re-check top redirect targets.
- Review cannibalisation between category, supplier and range pages.

## KPIs

- No major 404s from old URLs.
- Stable or improving clicks/impressions after migration volatility.
- More keywords in positions 4-20 moving toward page 1.
- More enquiries from product/range/category pages.
- More internal links to builder, floor levelling and category pages.
