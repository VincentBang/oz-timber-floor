Why this range matters in the Oz catalogue

Lumiere Ultra HD is a 7mm hybrid plank range with RCP core construction, ultra-matt PU finish and brushed timber-look surface.

It suits customers wanting a more refined hybrid visual while still using a floating-floor pathway. Final site and wet-area suitability should be confirmed before quote.
