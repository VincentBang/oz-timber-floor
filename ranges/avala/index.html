Questions about Avala

What type of flooring is Avala?

Avala is a hybrid flooring range. Oz Timber Floor can help confirm current product details, stock and project suitability before ordering.

What should I compare first in Avala?

Start with colour, board format, thickness and current stock timing, then confirm installation suitability for the project.

Can Oz Timber Floor supply Avala without installation?

Yes. Send the preferred colour, quantity and suburb so stock, batch and lead time can be checked.

Can Avala be quoted with installation?

Yes. Include the floor area, current floor type, suburb, access and timing so preparation and installation scope can be reviewed.

Can Oz Timber Floor help confirm product information for Avala?

Yes. Ask Oz Timber Floor to confirm current product information, warranty notes and project requirements before ordering.

What should I check before ordering Avala?

Confirm stock, batch, colour, floor condition, trims, transitions and installation suitability before ordering.
