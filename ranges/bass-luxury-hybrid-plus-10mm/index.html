Range overview

Luxury Hybrid PLUS 10mm is a thicker hybrid flooring range for a more substantial plank feel. Several colours use an 1815mm x 196mm plank format for a long-board visual. It suits projects where product finish, board thickness and a premium hybrid look matter.
